### MOST IMPORTANT THING TO REMEMBER

ğŸ‘‰ **If you see LISTEN + port + process â†’ service is running**

- `netstat` = **legacy, slower, basic info**
- `ss` = **modern, fast, detailed, flexible**

> Think of ss as netstat 2.0, designed for modern Linux.
> 

**Difference between ss & netstat:**

| Feature | `ss` | `netstat` |
| --- | --- | --- |
| Full form | **Socket Statistics** | **Network Statistics** |
| Status | Modern, recommended | Deprecated / legacy |
| Package | `iproute2` (default) | `net-tools` |
| Speed | Fast | Slow |
| Accuracy | More accurate (kernel-based) | Less accurate |
| Installation | Preinstalled | Manual install |
| Use today | âœ… Yes | âš ï¸ Avoid |
| Shows ports | Yes | Yes |
| Shows processes | Yes (`-p`) | Yes (`-p`) |
| Ubuntu preference | Default tool | Not preferred |
